Laporan Clustering

Tahapan Penyelesaian Tugas

1. Export Konten

Export konten beberapa group Whatsapp: Berasal dari beberapa GROUP yang berbeda untuk digabung menjadi satu file. File yang saya gunakan bernama data_group.txt

2. Konversi Data

Konversi data grub dari TXT ke CSV: Menggunakan aplikasi Excel dan kemudian dari aplikasi Excel export file tersebut menjadi file CSV. Hasil dari Konversi file menjadi data_group.csv setelah itu mengkonversi file data tersebut menjadi file TAR.

3. Pembersihan Data

Pembersihan Data: Mengambil angka, huruf dan tanda baca-tulis umum. Saya menggunakan code Python untuk melakukan filtering data pada file clean_data.py dan menghasilkan cleaned_data_group.csv.

4. Kode Program

Kode program scale_data.py dan many_kmeans_flow.py: Kode (scale_data.py) membaca data teks dari file CSV, mengubahnya menjadi angka (numerik) menggunakan metode TF-IDF, dan menyimpan hasilnya ke file CSV baru bernama scaled_data.csv.

Analisis dan Kesimpulan

Analisis

Hasil clustering menunjukkan bahwa untuk tiga kluster, istilah yang paling dominan adalah "856", "090", "ber" yang mungkin mencerminkan pola atau kategori tertentu dalam data, meskipun beberapa istilah tampak tidak lengkap. Untuk empat kluster, istilah seperti "eror", "gak", "32" menunjukkan adanya masalah atau ketidakpuasan, sementara untuk lima kluster, istilah "smart", "id", "ig", "event", dan "atau" menunjukkan fokus pada teknologi.

Kesimpulan

Kesimpulan dari hasil clustering ini adalah bahwa data yang dianalisis menunjukkan pola yang berbeda tergantung pada jumlah kluster yang digunakan. Pada tiga kluster, istilah yang muncul cenderung tidak lengkap dan mungkin mencerminkan kategori yang kurang jelas. Pada empat kluster, terdapat indikasi masalah atau ketidakpuasan yang diwakili oleh istilah seperti "eror" dan "gak". Sementara itu, pada lima kluster, istilah yang muncul lebih relevan dengan tema teknologi.
